Transcode Downloader

Adds a quality picker to the download action: grab the original, or a smaller server-side transcode, made with Jellyfin's own encoder (NVENC / QSV / VAAPI / software).

Output

How the transcoded file is encoded.

2 = stereo downmix, ideal for phones and tablets.

Quality presets

The download options offered to users. Min. source width hides a quality when the source is narrower than this, which prevents upscaling (e.g. a 1080p source won't show 4K). Leave it 0 to always offer the quality.
